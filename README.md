# Web Developer Portfolio

This is my personal portfolio website, built as a clean single-page frontend project to present who I am, what I build, and the tools I use.

The site focuses on clarity and usability: simple structure, readable content, responsive layout, and lightweight performance.

## Project Overview

This portfolio includes four main sections:

- Hero
- About
- Skills
- Projects

The goal is to keep the experience straightforward and professional without unnecessary elements.

## Tech Stack

- React 19
- Vite 8
- TypeScript
- Tailwind CSS 4
- ESLint
- Prettier
- Vitest
- React Testing Library

## Features

- Responsive single-page layout
- Accessible header navigation with mobile menu and skip link
- Reusable section and UI components
- Typed content data for navigation, skills, and projects
- Preloader with typewriter animation before the site reveals
